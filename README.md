# 001. API(Application Programming Interface)

> **두 소프트웨어 구성 요소가(클라이언트 / 서버) 서로 통신할 수 있게 하는 메커니즘**
> 

## 001-1. API TYPE

- GET : 데이터를 조회할때(Read)
- POST : 생성(Create), 변경(Update), 삭제(Delete)
